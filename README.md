**Задача проекта** marketplacce_onsights_and_AB_test - проанализировать данные быстрорастущего маркетплейса,
который уже нашел свою нишу, начал приносить прибыль.
Необходимо было понять, как чувствует себя продукт: кто приходит, кто остаётся, кто платит, кто уходит,
а также на каких каналах держится рост, какие категории товаров приносят деньги, а какие - нет. 


Информация предоставлена изначально в файлах csv отдельно по заказам, пользователям, событиям и расходам.


*Основные этапы анализа*:

- Знакомство с данными и предобработка;

- Обзор ключевых метрик маркетплейса;

- Оценка монетизации проекта и юнит-экономика;

- Поиск инсайтов и точек роста, сегментация и формулирование гипотез.


На основе полученных данных и рекомендаций команда маркетплейса приняла решение влиять на конверсию при помощи финансовых стимуляций,
дизайн теста разработали самостоятельно, предоставили данные для оценки состоятельности теста и итогов.


Для исследования были подключены библиотеки **Python**: pandas, matplotlib, seaborn, numpy, scipy.stats
